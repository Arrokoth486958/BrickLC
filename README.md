# Brick Launcher Core - A Minecraft Launcher Core
## Current Features
+ Microsoft/Mojang Authentication
+ Offline Account
+ Download Game
+ Download Libraries
+ Download Assets
+ Multisource support(like BMCLAPI)
+ Generate Batch file for launch
+ Launch Game
## Planed Features
+ Forge/Fabric download
+ XBox Authentication
+ Better API
+ Logging
+ Game Management
and more...
